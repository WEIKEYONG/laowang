速度： 

速度分为三种： 

基础速度：基础速度表示人物用一次移动动作能移动多远的距离，初始值为力量+敏捷+体积，单位是米。 

陆行速度：这种速度为角色在陆地上移动的速度，初始一般为基础速度，但是可能会因为某些能力而改变。 

飞行速度：这种速度为角色在天空中移动的速度，角色必须拥有特殊能力才可以飞行，在获得能力时，能力会说明角色的飞行速度。 



力量决定一部分攻击力与场景的互动。

敏捷决定防御力和闪避等动作。

体质决定生命值和对于病毒异常环境的抗性。

智力决定你的知识储备和法术伤害以及精神力

精神力决定使用法术的消耗。

意志力关于开启基因锁，每场轮回过后都可以增加10点意志力，完成某些支线也可以增加意志力。当受到大伤害或者濒死时可以申请开启基因锁。

注：普通人的初始四维均为2点。在这8点属性中你们拥有4点可以分配的属性并且必须要保证其中一项属性为1。1点属性并不影响除了投骰子以外的操作。


其他：

基因锁：在第一场过后可以在受到大伤害或者濒死时申请开启基因锁，基因锁会进行一个.rd(基因锁层数x100的判定)。

投骰子：基本上投骰子都是由主持人自己进行，为了保留轮回者的角色扮演体验不会在公共频道投骰子。

难度：难度与参加人数同等程度提高（唯心）

支线：支线的获取是假设当队里有一个成员拥有d级支线所兑换的东西时下一场会添加一定的c级支线，以此类推，当有成员兑换到b级物品并突破三阶基因锁时开启a级以及s级支线获取。
